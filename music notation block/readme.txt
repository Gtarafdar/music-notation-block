Add musical notation to your WordPress posts using ABC notation.
== Description ==
This WordPress block plugin allows users to easily add ABC music notation to their posts and pages. The plugin provides a user-friendly interface with the following features:

ABC Music Notation: Users can input ABC music notation code, which will be rendered as sheet music on the frontend.
Color Customization: Users can customize the color of the sheet music by selecting from a range of options.
Border Control: Users can adjust the border radius of the sheet music container to achieve their desired look.
Scale Fit: The plugin automatically scales the sheet music to fit the width of the content area, ensuring a clean and responsive display.

== Installation ==

Download the plugin zip file from the WordPress Plugin Directory or your preferred source.
In your WordPress dashboard, navigate to Plugins > Add New > Upload Plugin.
Select the zip file and click "Install Now".
Once installed, activate the plugin.

== Usage ==

In the WordPress block editor, add a new block and search for "ABC Music Notation".
In the block settings, enter your ABC music notation code.
Customize the color, border radius, and scale fit settings as desired.
Publish your post or page, and the sheet music will be displayed on the frontend.

== ABC Music Notation Resources ==
For more information on ABC music notation and to find existing notation, visit the ABC Notation website.
